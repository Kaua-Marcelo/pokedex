POKEDEX

Overview

Pokédex is a web application developed with a focus on organization, clarity, and good development practices. The project was structured in a simple and objective way, prioritizing clean code and a consistent base for future evolution. During development, basic planning, implementation, and validation processes were followed, ensuring a functional and well-structured delivery. The application was built using HTML, CSS, JavaScript, and Figma.



The development was carried out by a team of four developers, who worked collaboratively throughout the entire project.
